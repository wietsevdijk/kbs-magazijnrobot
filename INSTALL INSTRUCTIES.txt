Extract dit rar bestand op een locatie die je onthoud ;)

mysql connecter module toevoegen:
Ga naar File -> Project Structure -> modules -> plus tekentje (onder module SDK) -> JARs or Directories -> selecteer ojdbc8.jar

java database driver toevoegen:
Ga naar File -> Project Structure -> Libraries -> plus tekentje -> selecteer mysql-connector-j-8.0.33.jar
